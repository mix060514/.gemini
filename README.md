# Gemini CLI

This is the configuration and core directory for the Gemini CLI agent.

## Structure

- `GEMINI.md`: Contains the context and memories for the Gemini agent.
- `settings.json`: Configuration settings.
- `skills/`: Specialized skills for the agent.
- `commands/`: Custom commands.
- `antigravity/`: (Internal or specific functionality)

## Usage

This project is used to manage and extend the capabilities of the Gemini interactive CLI.
